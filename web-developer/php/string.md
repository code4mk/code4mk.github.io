### string

* single quote

```php
$name = 'kamal'
```

```php
echo 'kamal is here \n new line';
//  kamal is here \n new line
```

* double quote

```php
$name = "kamal"
```

```php
echo "kamal is here \n new line";
//  kamal is here
//  new line
```
