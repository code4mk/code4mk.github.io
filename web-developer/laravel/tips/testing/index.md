
## testing

# configure xml file

```bash
# bash cli
./vendor/bin/phpunit --configuration /path/to/laravels/phpunit.xml
```

# run

```bash
# all file
./vendor/bin/phpunit
# specific file
./vendor/bin/phpunit tests/Unit/AppTest.php
```

# optional

* global install

```bash
composer global require phpunit/phpunit
```
