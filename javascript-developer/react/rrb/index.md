## What is RRB
![rrb](https://github.com/code4mk/react-redux-boss/raw/master/src/assets/rrb1.png)

RRB is a `react-redux-boss` mini boilerplate of react.

## Features

* React
* React Router Dom
* Redux (coming soon)
* mobx
* Axios
* Bulma

## How use RRB

``` bass
  Clone RRB project
  yarn install or npm install
  npm run dev
```
