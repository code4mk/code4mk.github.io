## ready

jquery library loading after ready . jquery has 4 types ready function .

* 1

```js
$(document).ready(function() {
   // jQuery code goes here
});
```

* 2

```js
$(function(){
  // jQuery code goes here
})(jQuery);

```

* 3

```js
$(function(){
  // jQuery code goes here
});
```

* 4

```js
$(window).load(function(){
  // jQuery code goes here
});
```
