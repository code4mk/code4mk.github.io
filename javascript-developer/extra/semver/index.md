
 ~ `version [Major].[Minor].[Patch]`

* Major = breaking change
* Minor = new feature
* Patch = bugfix

| ranges     | descriptions     |
| :------------- | :------------- |
| ^1.2.1       | 1.2.1< x < 2.0.0       |
|^0.2.1|0.2.1< x < 0.3.0|
|^0.0.1|0.0.1< x < 0.0.2|
|~1.2.1|1.2.1< x < 1.3.0|
|~0.2.1|0.2.1< x < 0.3.0|
