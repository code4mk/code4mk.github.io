## arithmatic operators

| symbols     | Description     |
| :------------- | :------------- |
| `+`       | plus      |
|`-`|minus|
|`*`|multiplacation|
|`/`|division|
|`%`|modulus / remainder|
|`++a`|pre increment|
|`a++`|post increment|
|`--a`|pre decrement|
|`a--`|post decrement|
|`=`|assign|

### order operator

```bash
exponents and roots
multiplication and division

```
