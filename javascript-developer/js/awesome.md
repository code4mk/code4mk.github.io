

## Fundamental objects

### Object

* `object` is a collections of data . objects represts by `{ ... }` or array objects `[{},{}]`

```js
const user = {
  name: 'kamal',
  age: 21
}
```
#### Array type objects

```js
const users = [
  { name: 'kamal' , age: 21},
  { name: 'jamal', age: 25}
]
```
